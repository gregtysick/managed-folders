# Task System

ManagedFolders uses a four-part task system:

- Task Radar
- Project Manager
- Now
- Today

Scribe supports the task system with evidence and history.

## Simple Rule

- If it is only a possibility, keep it in Task Radar until approved.
- If it is approved managed work, keep it in Project Manager.
- If it should be done now, surface it in Now.
- If it should stay visible today, surface it in Today.

## Work Selection

When the operator asks what to work on now, Executive should review Project Manager first, then current state, open loops, recent evidence, and visible Now/Today surfaces before recommending the next workstream.
