# ManagedFolders 1.0

Status: historical  
Released: 2026-05-31

ManagedFolders `1.0` introduced the public mirror as a self-contained management-system project.

## Release Focus

- Start the public ManagedFolders repository.
- Describe the idea that a folder can be more than passive storage.
- Establish the first public-safe documentation shape.
- Begin separating public roadmap material from private operating workspaces.

## Release Principle

Version `1.0` is the seed release.

It starts the public record without exposing private workspace details.
