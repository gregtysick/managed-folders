# ManagedFolders 2.0 Preview

Status: historical preview  
Released: 2026-06-10

ManagedFolders `2.0-preview` introduced the public shape of the operating model before the full `2.0` release note.

## Preview Focus

- Introduce Executive as the current operating layer.
- Introduce Scribe as the evidence and history layer.
- Introduce Admin / Knowledge as durable reference space.
- Introduce Task Radar as candidate-work detection.
- Introduce Project Manager as approved-work ownership.

## Preview Principle

The preview made the major 2.0 layer model visible while keeping implementation details private.
