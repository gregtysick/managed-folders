# Documentation

Documentation explains the ManagedFolders system in public-safe language.

## Role

- explain the whole system at a high level
- describe subsystems such as Executive, Task Radar, Project Manager, Scribe, and Registry
- keep public readers oriented without exposing private implementation notes

## Public Boundary

Documentation in this repo should be generic and easy to scan.

Detailed private operating records, client-specific examples, and local paths stay out of the public mirror.
